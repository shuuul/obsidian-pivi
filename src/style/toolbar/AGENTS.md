# `src/style/toolbar/` — Chat toolbar CSS

Styles for model, mode, MCP, thinking, permission, and external-context toolbar controls.

## Rules

- Dropdowns must be usable with keyboard/focus, not just `:hover`.
- Keep selector names scoped and consistent with toolbar component class names.
- Use shared design tokens (`--obsius2-brand`, `--obsius2-error`, Obsidian variables).
- Import every new file through `../index.css`.
